# 🪙 Proyecto: Sopas_token (SPV)

Este proyecto consiste en la creación, despliegue y puesta en mercado de una criptomoneda propia sobre la red de pruebas de Ethereum (Sepolia).

## 🛠️ Requisitos previos
* [cite_start]**MetaMask:** Instalado en el navegador.
* [cite_start]**Red Sepolia:** Activada en la configuración de redes de prueba.
* [cite_start]**Saldo (Gas):** Necesitarás Sepolia ETH de un Faucet (ej. Google Faucet).

## 🚀 Despliegue del Contrato
1. [cite_start]Abrir el código en **Remix IDE** utilizando el estándar **ERC20 de OpenZeppelin**.
2. [cite_start]**Compilar:** Usar `Solidity Compiler`.
3. [cite_start]**Desplegar:** En la pestaña `Deploy & Run`, seleccionar `Injected Provider - MetaMask`.
4. [cite_start]**Confirmar:** Pagar el gas en la ventana emergente de MetaMask.
